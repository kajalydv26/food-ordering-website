# 🍔 Food Ordering Website

A full-stack food ordering web application designed to provide a seamless and user-friendly experience for browsing food items, managing cart operations, and placing orders. The application demonstrates strong integration of frontend and backend technologies along with efficient database management.

---

## 🚀 Key Features

- 🛒 Interactive Add to Cart functionality with dynamic updates  
- 📋 Well-structured and categorized food menu display  
- 🧾 Smooth order placement system with data storage  
- 🔄 Real-time interaction between frontend and backend  
- 📱 Responsive and user-friendly interface for better user experience  
- 💾 Efficient handling of user and order data  

---

## 🛠️ Technologies Used

### 🔹 Frontend
- HTML5 for structuring web pages  
- CSS3 for styling and responsive design  
- JavaScript for dynamic behavior and interactivity  

### 🔹 Backend
- PHP for server-side logic and handling user requests  
- Integration of frontend with backend for seamless data flow  

### 🔹 Database
- MySQL for storing and managing application data  
- SQL queries for data insertion, retrieval, and management  

### 🔹 Development Tools
- XAMPP for local server environment  
- VS Code for development and debugging
